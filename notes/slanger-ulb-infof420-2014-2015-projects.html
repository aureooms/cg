  • Draining a polygon. Imagine a polygon P containing water. Is it possible to empty all the water by piercing one hole and rotating the polygon? Or is it necessary to pierce more than one hole (and then what is the minimum number of holes needed)? CGTA Volume 47, Issue 2, Part C, February 2014, Pages 316â€“328 pdf
  • Highway Hull. A highway H is a line in the plane on which one can travel at a greater speed than in the remaining plane. One can choose to enter and exit H at any point. The highway time distance between a pair of points is the minimum time required to move from one point to the other, with optional use of H. The highway hull H(S,H) of a point set S is the minimal set containing S as well as the shortest paths between all pairs of points in H(S,H), using the highway time distance. CGTA Volume 43, Issue 2, February 2010, Pages 115â€“130
  • Compatible Triangulations. Given two planar n point sets, is it always possible to triangulate them both so that the two resulting graphs are isomorphic? This question is explored in this article: Theoretical Computer Science, Volume 296, Num. 1, pp. 3 - 13, 2003. (ps).
  • String-wrapped Rotating Disks. Given a collection of disks in the plane, each labeled + or -, can we always wrap a string around the disks so that when it moves, the + disks move clockwise and the - disks move counterclockwise? Lecture Notes in Computer Science Volume 7579, 2012, pp 65-78 (pdf).
  • Zonoids. A Zonoid can be seen as an interpolation between a convex hull and the center of mass of a set of points. They have a very interesting structure. http://cg.scs.carleton.ca/~morin/publications/depth/zonoidd-cgta.pdf http://portal.acm.org/citation.cfm?id=1290772
  • From decision to optimization. If there is an algorithm that solves a decision problem (e.g., can we place this statue in a box of size x), is it possible to solve the corresponding optimization problem (what is the smallest box that can contain this statue) almost as fast?
